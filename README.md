# Frontend IT Company Project

## Overview
This is a responsive and professional IT company website developed as part of a front-end development internship task. The project represents a fictional IT services company named **NexoraTech Solutions**.

## Features
- Professional UI/UX design
- Responsive layout for mobile, tablet, and desktop screens
- Dark mode support
- Hamburger navigation for small screens
- Detailed Services section describing all services offered
- Testimonials and FAQ sections
- Contact form with client-side validation

## Pages Included
- **Home** – Overview, services highlight, testimonials
- **About** – Company mission, vision, values
- **Services** – Detailed descriptions of IT services
- **Contact** – Professional contact UI with form validation

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
